## Fix audit issues

- delete `node_modules`
- delete `package-lock.json`
- run `npm install`

This pulls all dependencies again, meaning that all versions of dependencies marked with ^, ~ etc. will pull newer versions if available.

> `npm audit fix` should perform the same task as above.
